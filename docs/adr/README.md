# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for the Finance Tracker app.

ADRs capture:

- Context: what we knew at the time
- Decision: what we chose
- Consequences: tradeoffs and follow-ups

## Conventions

- File naming: `NNNN-short-title.md`
- Status: `Proposed` | `Accepted` | `Rejected` | `Superseded`
- When a decision changes: add a new ADR and mark the old one `Superseded`.

## Template

Use `docs/adr/template.md` for new ADRs.
